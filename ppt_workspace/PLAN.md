## 三、逐页规划

1. 封面：用工业蓝图风格展示薄膜双拉线、在线检测、Agent 控制台，建立“研发优化平台”的整体感。
2. 结论先行：说明模式可行，但必须是“受控在线研发优化平台”，不是 LLM 直接控设备。
3. 为什么原始设想需要收紧：列出 5 个不合理设计和修正方向。
4. 总体架构：人机协同层、Agent 编排层、Skill 流程层、MCP 接口层、设备与数据层。
5. MVP-1 安全闭环：七步链路，从采集快照到形成可学习样本。
6. MCP 层怎么做：六个 MCP 的职责、第一版交付和验收。
7. Skill 层怎么做：快环、深环、诊断审计、仿真后置的边界。
8. Agent 层怎么做：总编排、工艺工程师、研发工程师、安全解释、操作员协同。
9. 开发路线与验收：P0-P4 阶段，强调先闭环后智能。
10. 最终落地蓝图：把关键设计原则收束成项目开发行动清单。

## 五、技术规范

- PPTX 尺寸：16:9。
- 保证所有图形可编辑，流程图和架构图使用 PPT shape。
- 不使用营销式大空话，所有页面围绕“如何落地开发”。
- 每页最多一个视觉主角，文字密度服务于工程决策。
